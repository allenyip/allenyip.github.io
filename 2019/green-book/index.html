看《绿皮书》

Written by Allen Yip with ♥ on March 11, 2019 in Movie

本片基于真实故事改编，虽然与真实故事存在着一些令人反感的出入，但仅从电影本身来看，它对我来说是一部久违的好电影。

电影讲述的是一位优雅的黑人音乐家聘请一个粗鲁的白人司机，到种族歧视严重的南部巡回演出的故事。一路上欢声笑语，有血有泪，印象比较深刻的有以下几个地方：

  1. 汽车在田边故障，司机停下来修车，音乐家在汽车后座坐着，与田里劳作的黑人们互相对望，形成鲜明对比。
  2. 音乐家教司机写信，司机教音乐家如何正确的吃炸鸡。
  3. 尽管是世界知名的音乐家，受邀来到表演地仍然没法受到平等对待，音乐家一直坚持维护好自己的尊严。
  4. 司机说：孤独的人都害怕迈出第一步。
  5. 音乐家说：自己不像白人，不像黑人，不够男人，那我到底算什么人？
  6. 最后在一间黑人酒吧弹奏肖邦的练习曲，接受并找回了自我。
